# OrangeHRM Cypress Automation - Final Project SanberCode

## Description

This repository contains the final project for the SanberCode QA Bootcamp. It is an automated End-to-End (E2E) testing framework built for the [OrangeHRM](https://opensource-demo.orangehrmlive.com/) web application. The framework leverages **Cypress** combined with **Cucumber (BDD)** to execute test scenarios efficiently while maintaining highly readable test cases. The project implements the **Page Object Model (POM)** design pattern for better maintainability and reusability.

## Technologies Used

- **Cypress** (v14.0.2)
- **Cucumber (BDD)** - For writing tests in Gherkin syntax.
- **JavaScript** - Core programming language used for test scripts.
- **Mochawesome Reporter** - For generating comprehensive, visually appealing HTML test reports.

## Features Tested

The automation suite covers the following modules of the OrangeHRM application:

- **Login**: Verification of valid/invalid login and authentication flows.
- **Forgot Password**: Password reset functionality.
- **Dashboard**: Verification of dashboard widgets (e.g., Time at Work, Buzz, Leave Today).
- **Directory**: Searching and filtering directory records.

## Setup & Installation

### Prerequisites

- Node.js (v16 or higher recommended)
- npm (Node Package Manager)

### Installation

1. Clone this repository to your local machine:
   ```bash
   git clone https://github.com/mas-andri/sanbercode-final-project-orangehrm
   ```
2. Navigate to the project directory:
   ```bash
   cd sanbercode-final-project-orangehrm
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Running the Tests

This project includes several custom npm scripts to run tests in different modes and browsers.

### Cypress Interactive Mode (Test Runner)

To open the Cypress UI and run tests interactively:

```bash
npx cypress open
```

### Running Specific Modules

You can run specific test modules using the following scripts:

- **Login Tests**: `npm run cy:run:login`
- **Forgot Password Tests**: `npm run cy:run:forgot-password`
- **Dashboard Tests**: `npm run cy:run:dashboard`
- **Directory Tests**: `npm run cy:run:directory`

### Regression & Smoke Tests

- **Regression Tests** (Runs all features):
  - Chrome: `npm run cy:run:regression:headless:chrome`
  - Firefox: `npm run cy:run:regression:headless:firefox`
- **Smoke Tests** (Runs Login & Dashboard only):
  - Chrome: `npm run cy:run:smoke:headless:chrome`
  - Firefox: `npm run cy:run:smoke:headless:firefox`

## Test Reports

This project uses `cypress-mochawesome-reporter` to generate HTML reports automatically after a test run.

- Reports are generated by default in the `cypress/reports/html` directory.
- The report includes embedded screenshots for any failed tests.
- Open `index.html` in your browser to view the detailed test execution results.
